# Sales Copilot Mobile - 移动销售CRM智能体应用

## 概述
面向销售人员的移动端CRM智能体应用，采用玻璃拟态(Glassmorphism)设计风格，支持深浅色主题。核心交互通过语音输入（底部麦克风按钮）完成，无键盘入口。

## 设计规范

### 色彩系统
- **主色**: `#FF7A00` (橙)
- **强调色**: `#0D8F8C` (青)
- **深色背景渐变**: `linear-gradient(180deg, #0F1424 0%, #161B2C 100%)`
- **正文**: `#E6E9F4`
- **卡片**: `rgba(255,255,255,0.06)` + `border 1px rgba(255,255,255,0.10)` + 圆角 `12px`

### 字号体系
- 标题: 14pt / 600 weight
- 正文: 11pt
- 辅助: 9.5pt

### 全局UI组件
1. **顶栏**: 左侧 `←` 返回、中间页面名称、右侧 `⋯` 更多
2. **底部麦克风**: 橙色圆形按钮 (56px直径)，按住录音 - 唯一输入入口
3. **无键盘图标**: 禁止任何键盘/文本输入入口

## 数据架构

### 数据源
- **Dataverse OOB表复用**: Account, Opportunity, Activity (activitypointer + subtypes)
- **自定义表**: AgentLog (AI智能体交互日志)

### 数据模型
待用户提供三屏具体需求后确定完整字段结构。初步规划：

1. **Account** - 客户账户
2. **Opportunity** - 销售机会
3. **Activity** - 活动记录 (任务/电话/会议)
4. **AgentLog** - 智能体操作日志

## 屏幕结构

应用包含三个互通屏幕，具体功能待用户后续提供详细需求。

## 技术要点

1. **语音优先交互**: 所有输入通过底部麦克风完成
2. **玻璃拟态风格**: 半透明卡片 + 模糊背景 + 精细边框
3. **移动竖屏优化**: 针对手机竖屏设计布局和交互
4. **深浅色支持**: 完整的明暗主题切换
5. **Motion动效**: 流畅的页面转场和微交互
